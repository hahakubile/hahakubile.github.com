Ubuntu Config Locale With zh_CN.UTF-8

Feb 24th, 2014

查看系统内安装的locale

locale -a

查看当前系统语言环境

locale

安装语言包: zh_CN.utf8

cd /usr/share/locales

./install-language-pack zh_CN

设为默认

sudo vim /etc/default/locale

修改为:

LANG="zh_CN.UTF-8"

LANGUAGE="zh_CN:zh"

重启shell，OK
